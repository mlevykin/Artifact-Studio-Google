# 🌍 Уровень 1: Системный контекст (System Context)

## Описание системы
**AI Studio Build** — это полнофункциональная среда для разработки и визуализации артефактов. Система позволяет пользователям взаимодействовать с LLM для создания кода, документации и сложных диаграмм в реальном времени.

## Основные возможности
- Генерация и редактирование артефактов (Markdown, Mermaid, SVG, Banana JSON).
- Интеграция с внешними API (Gemini).
- Управление проектами и файловой системой.
- Высококачественная визуализация с поддержкой зума и панорамирования.

## Внешние зависимости
- **Gemini API**: Интеллектуальное ядро системы.
- **Browser Runtime**: Среда исполнения и рендеринга.
