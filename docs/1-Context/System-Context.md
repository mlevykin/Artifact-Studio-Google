# 🌍 Уровень 1: Системный контекст (System Context)

Этот документ описывает приложение как «черный ящик» и его взаимодействие с внешним миром.

## Описание
Приложение представляет собой интеллектуальную среду для генерации, редактирования и визуализации артефактов (Markdown, Mermaid, SVG, Banana JSON).

## Внешние взаимодействия
- **Пользователь**: Взаимодействует через чат и панель артефактов.
- **LLM (Gemini API)**: Используется для генерации контента и логического разбора запросов.
- **Браузер**: Среда исполнения и рендеринга.
