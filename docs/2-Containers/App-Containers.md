# 📦 Уровень 2: Контейнеры (Containers)

Раскрытие внутренней структуры системы на уровне исполняемых единиц.

## Основные контейнеры

1.  **Frontend (React SPA)**:
    - Основной интерфейс пользователя.
    - Управляет состоянием артефактов и чата.
    - Ссылка: [[UI-Components-Map|Карта компонентов UI]]
2.  **Backend (Express Server)**:
    - Проксирует запросы к API.
    - Управляет статическими файлами.
3.  **Storage (Browser LocalStorage / IDB)**:
    - Хранение истории сессий и настроек.

## Схема взаимодействия
`User -> [Frontend] <-> [Backend] <-> [Gemini API]`
