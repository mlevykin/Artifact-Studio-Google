# 📦 Контейнер: Backend (Express Proxy)

## Описание
Серверная часть приложения на Node.js/Express. Служит прокси-слоем для взаимодействия с внешними API и раздачи статического контента в продакшене.

## Основные функции
- Проксирование запросов к Gemini API.
- Обработка CORS и заголовков безопасности.
- Раздача файлов из директории `dist/`.
