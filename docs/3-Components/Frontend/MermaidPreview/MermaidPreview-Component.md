# 🧩 Компонент: MermaidPreview (Renderer)

## Описание
Специализированный компонент для рендеринга диаграмм Mermaid.js.

## Особенности
- Поддержка различных стилей (включая [[3-Components/Frontend/BananaRenderer/BananaRenderer-Component|Paper Banana]]).
- Интерактивное масштабирование внутри [[3-Components/Frontend/ZoomableContainer/ZoomableContainer-Component|ZoomableContainer]].
- Поддержка пошагового отображения в режиме презентации.
