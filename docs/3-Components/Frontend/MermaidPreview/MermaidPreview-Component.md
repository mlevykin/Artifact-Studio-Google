# 🧩 Компонент: MermaidPreview (Renderer)

## Описание
Специализированный компонент для рендеринга диаграмм Mermaid.js.

## Особенности
- Поддержка различных стилей (включая [[Banana-JSON-Contract|Paper Banana]]).
- Интерактивное масштабирование внутри [[ZoomableContainer-Component]].
- Поддержка пошагового отображения в режиме презентации.
