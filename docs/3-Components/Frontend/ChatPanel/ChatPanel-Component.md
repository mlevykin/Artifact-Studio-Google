# 🧩 Компонент: ChatPanel

## Описание
Интерфейс для взаимодействия пользователя с ИИ. Обеспечивает отправку сообщений, отображение истории диалога и индикацию процесса генерации.

## Функциональность
- Поле ввода с поддержкой многострочного текста.
- Отображение сообщений пользователя и ответов ИИ.
- Поддержка вложений и контекстных подсказок.
- Интеграция с `ArtifactPanel` для автоматического открытия сгенерированных артефактов.
