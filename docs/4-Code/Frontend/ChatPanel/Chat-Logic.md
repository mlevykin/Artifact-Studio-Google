# 💻 Код: Логика чата

## Описание
Реализация взаимодействия с Gemini API, обработка стриминга ответов и управление историей сообщений.

## Основные функции
- `sendMessage`: Отправка запроса с учетом текущего контекста проекта.
- `processStream`: Разбор потоковых данных от LLM и обновление UI.
- `handleArtifacts`: Обнаружение и парсинг артефактов в ответах ИИ.
