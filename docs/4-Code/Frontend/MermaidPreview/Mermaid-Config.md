# 💻 Код: Конфигурация Mermaid

## Описание
Настройки и кастомизация библиотеки Mermaid.js для соответствия эстетическим стандартам приложения.

## Параметры
- `theme`: Использование кастомной темы "Paper Banana".
- `flowchart`: Настройки скругления углов и толщины линий.
- `securityLevel`: Настройки безопасности для рендеринга SVG.
